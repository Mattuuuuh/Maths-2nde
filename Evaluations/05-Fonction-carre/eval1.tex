\exe{4, difficulty=2}{
	Considérons la courbe représentative de la fonction $f(x) = 3-2x$ et le point $A(2;1)$.
	Le but de l'exercice est de trouver le point de $\C_f$ le plus proche de $A$.
	\begin{enumerate}
		\item
		Grapher $\C_f$ sur le domaine $[0 ; 4]$ et placer le point $A$ dans un repère orthonormé.
		Le point $A$ appartient-il à $\C_f$ ? Justifier.
		\item
		Montrer qu'un point de $\C_f$ est de la forme $B(x ; 3-2x)$, avec $x\in\R$.
		\item
		Montrer que la distance au carré de $B$ à $A$ est égale à
			\[ AB^2 = \dfrac15 (5x-6)^2 + \dfrac45. \]
		\emph{Formule de la distance au carré : $AB^2 = (x_A - x_B)^2 + (y_A - y_B)^2$.}
		\item
		Conclure que le point de $\C_f$ le plus proche de $A$ est $B(\frac65 ; \frac35)$.
		Placer $B$ dans le repère déjà construit.
	\end{enumerate}
}{exe:proj-affine}{
	fait un peu rapidement, à vérifier.
}
