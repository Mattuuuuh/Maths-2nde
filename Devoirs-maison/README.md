# Template

Le dossier `00-DM-Template/` donne un exemple de devoir maison vierge.

Pour compiler :
```
mkdir out adr
python3 generate.py
```

Pour fusionner :
```
pdftk $(ls out/*.pdf) cat output merged.pdf
```
